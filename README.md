# Myntra Clone

A responsive front-end clone of the Myntra website built using **HTML5** and **CSS3**.

## 🚀 Features

* Responsive Navigation Bar
* Search Bar
* Hero Banner Section
* Product Categories
* Fashion Product Cards
* Footer Section
* Clean User Interface

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

```text
myntra-clone/
│
├── index.html
├── style.css
├── images/
└── README.md
```
